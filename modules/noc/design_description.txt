Module: noc

Target: 32-bit RISC-V SoC — interconnect

Network-on-chip router for scalable RV32 multi-core or multi-cluster SoC.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- in_valid [P-1:0]  input  — per port
- in_data [P-1:0][63:0]  input  — flit
- out_valid [P-1:0]  output  — output
- out_data [P-1:0][63:0]  output  — output
- in_ready [P-1:0]  output  — output
- out_ready [P-1:0]  input  — input

Behavior:
- XY or mesh routing; wormhole or store-and-forward.
- Carries cache-coherent or IO traffic between tiles.
