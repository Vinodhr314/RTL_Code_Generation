Module: compression_engine

Target: 32-bit RISC-V SoC — accelerator

Hardware compressor/decompressor (LZ77 or run-length) for IO bandwidth.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- mode  input  — compress/decompress
- in_valid  input  — input
- in_data [31:0]  input  — input
- in_last  input  — input
- out_valid  output  — output
- out_data [31:0]  output  — output
- out_last  output  — output
- busy  output  — output

Behavior:
- Streams data to/from packet buffer or DMA.
- Reduces off-chip traffic for RV32 workloads.
