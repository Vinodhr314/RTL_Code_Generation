Module: gp_dma_gen_1

Target: 32-bit RISC-V SoC — dma

Generation-1 GP DMA with dual channels and linked descriptors.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- ch0/ch1  start,  — src, dst, len, done, irq
- mem  AXI-lite  — master port

Behavior:
- Two independent channels; descriptor ring in RV32 memory.
- Superset of gp_dma for higher throughput.
