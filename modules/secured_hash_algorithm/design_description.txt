Module: secured_hash_algorithm

Target: 32-bit RISC-V SoC — crypto

SHA-256 secure hash accelerator.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- start  input  — input
- block [511:0]  input  — input
- digest [255:0]  output  — SHA-256
- done  output  — output
- busy  output  — output

Behavior:
- FIPS 180-4 SHA-256 one block per cycle after pipeline fill.
- Used for secure boot signature verify.
