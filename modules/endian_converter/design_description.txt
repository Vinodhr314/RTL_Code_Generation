Module: endian_converter

Target: 32-bit RISC-V SoC — datapath

Byte/half-word endian swap for RV32 BE/LE bus bridging.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- din [31:0]  input  — input
- dout [31:0]  output  — output
- mode [1:0]  input  — B/H/W swap

Behavior:
- Converts 32-bit word between little-endian core and big-endian peripheral.
- No clock; pure combinational swap network.
