Module: fpu_double_precision

Target: 32-bit RISC-V SoC — accelerator

IEEE-754 double-precision FPU for RV32D-style offload (64-bit).

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- valid  input  — input
- op [2:0]  input  — input
- a [63:0]  input  — input
- b [63:0]  input  — input
- result [63:0]  output  — output
- ready  output  — output
- flags [4:0]  output  — output

Behavior:
- Double-precision add/sub/mul/div multi-cycle unit.
- Results staged to GP regs via memory map.
