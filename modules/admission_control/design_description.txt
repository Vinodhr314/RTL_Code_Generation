Module: admission_control

Target: 32-bit RISC-V SoC — network

Rate/token-bucket admission control for inbound/outbound SoC traffic.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- pkt_valid  input  — input
- class [3:0]  input  — input
- accept  output  — output
- drop  output  — output
- tokens [15:0]  input  — config

Behavior:
- Per-class token buckets; drops when exceeded.
- Protects RV32 from network floods.
