- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Verification status: PASS

================================================================================
2. FUNCTIONAL REQUIREMENTS
================================================================================
FR-001: Module shall implement CRC-32 Ethernet generator (polynomial 0x04C11DB7), byte-serial..
FR-002: Accept block/key on valid/start; assert done when digest/ciphertext ready.
FR-003: Output width and algorithm per locked spec and RTL header comments.
FR-004: init resets crc to 32'hFFFFFFFF. Each valid byte updates CRC per Ethernet CRC-32 (reflected). done pulses one cycle after each valid byte.

5. FUNCTIONAL BEHAVIOR
================================================================================
