Module: sch

Target: 32-bit RISC-V SoC — system

Hardware scheduler / arbiter for accelerator job queues in the SoC.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- req [N-1:0]  input  — job requests
- grant [N-1:0]  output  — one-hot grant
- priority [7:0]  input  — config

Behavior:
- Round-robin or priority scheduling among N accelerator clients.
- Used by SPU, VPU, DMA job submitters.
