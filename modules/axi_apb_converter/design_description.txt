Module: axi_apb_converter
Document: Design Specification for Verification (DSV)
Revision: 1.0
Target: 32-bit RISC-V SoC
Protocol/Standard: ARM AMBA AXI4 / AXI4-Lite

================================================================================
1. OVERVIEW
================================================================================
AXI4-Lite slave to APB master bridge for single-beat peripheral access.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Verification status: PASS

================================================================================
2. FUNCTIONAL REQUIREMENTS
================================================================================
FR-001: Module shall implement AXI4-Lite slave to APB master bridge for single-beat peripheral access.
FR-002: Slave shall accept AXI4 write/read bursts per channel handshake rules.
FR-003: awready/wready/arready shall follow RTL timing (typically immediate or registered).
FR-004: bresp/rresp shall return OKAY (2'b00) on successful transfers.
FR-005: Bridge: AXI write (AW+W) -> APB SETUP then ACCESS phase write -> B response. AXI read (AR) -> APB SETUP then ACCESS phase read -> R response

================================================================================
4. TIMING, CLOCKING & RESET
================================================================================
Clock domain: single pclk (APB).
Reset: presetn active-low asynchronous assert, synchronous deassert.
APB timing: SETUP cycle (psel=1, penable=0) then ACCESS cycle (psel=1, penable=1).
DV note: Sample prdata/pready on posedge pclk during ACCESS phase.

================================================================================
5. FUNCTIONAL BEHAVIOR
================================================================================
Bridge: AXI write (AW+W) -> APB SETUP then ACCESS phase write -> B response. AXI read (AR) -> APB SETUP then ACCESS phase read -> R response. pclk = aclk, presetn = aresetn.

RTL header notes:
- AXI4-Lite to APB bridge
- Translates AXI single-beat writes/reads to APB SETUP+ACCESS transactions
- Verilog-2005 synthesizable
- AXI4-Lite slave — write address
- AXI4-Lite slave — write data
- AXI4-Lite slave — write response
- AXI4-Lite slave — read address
- AXI4-Lite slave — read data

FSM states (from RTL localparams):
- ST_IDLE → ST_AW_W

  TS-02: AXI write:
  TS-03: DUT only asserts awready/wready after seeing awvalid/wvalid.

  TS-07: Wait for write response — don't set bready until bvalid is seen
