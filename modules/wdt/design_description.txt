Module: wdt

Target: 32-bit RISC-V SoC — system

Watchdog timer resetting RV32 on software hang.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- kick  input  — pet watchdog
- timeout [31:0]  input  — reload value
- wdt_rst  output  — system reset
- irq  output  — warning before reset

Behavior:
- Counts down from reload; kick must occur before zero.
- Optional windowed watchdog mode.
