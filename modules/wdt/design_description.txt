- kick (input, 1, —): Input signal, width 1.
- timeout (input, [31:0], —): Input signal, width [31:0].
- wdt_rst (output, 1, —): Output signal, width 1.
- irq (output, 1, —): Interrupt request to CPU (typically level or one-cycle pulse).
