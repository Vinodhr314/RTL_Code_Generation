================================================================================
1. OVERVIEW
================================================================================
Custom compressed-ISA decode assist for fused add (opcode 7'h7B)..

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Verification status: PASS

================================================================================
2. FUNCTIONAL REQUIREMENTS
================================================================================
FR-001: Module shall implement Custom compressed-ISA decode assist for fused add (opcode 7'h7B)..
FR-002: Outputs shall be purely combinational with no clock or reset dependency.
FR-003: All port widths shall match the locked specification exactly.
FR-004: Functional relation: When valid and inst[6:0]==7'h7B: rd = rs1 + rs2, rd_valid=1 next cycle. Otherwise rd=0, rd_valid=0..
FR-005: When valid and inst[6:0]==7'h7B: rd = rs1 + rs2, rd_valid=1 next cycle. Otherwise rd=0, rd_valid=0.
