Module: fifo

Target: 32-bit RISC-V SoC — infrastructure

Synchronous FIFO buffer for clock-domain-safe streaming between RV32 subsystems.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- wr_en  input  — input
- rd_en  input  — input
- din [31:0]  input  — write data
- dout [31:0]  output  — read data
- full  output  — output
- empty  output  — output
- count [ADDR_W-1:0]  output  — occupancy

Behavior:
- Parameterized depth (default 16). First-word-falls-through or standard.
- Used between DMA, MAC, and packet buffers.
