Module: bch_encoder

Target: 32-bit RISC-V SoC — algorithm

BCH forward error correction encoder for flash/link protection.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- data_in [7:0]  input  — input
- valid  input  — input
- codeword [N-1:0]  output  — output
- done  output  — output

Behavior:
- Configurable BCH(n,k,t); adds parity bits to data block.
- Paired with BCH decoder in storage subsystem.
