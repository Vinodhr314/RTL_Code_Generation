Module: pp_ext_mem

Target: 32-bit RISC-V SoC — memory

Processing pipeline external memory interface stage.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- addr [31:0]  input  — input
- wdata [31:0]  input  — input
- wen  input  — input
- ren  input  — input
- rdata [31:0]  output  — output
- ready  output  — output

Behavior:
- Bridges PP datapath to external SRAM/DRAM port with registered interface.
- Supports single-cycle or wait-state memory.
