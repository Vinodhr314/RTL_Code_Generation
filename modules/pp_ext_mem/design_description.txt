3.2 Signal Descriptions
- clk (input, 1, —): System clock; all sequential logic sampled on rising edge unless noted.
- rst_n (input, 1, active-low): Asynchronous active-low reset; deassert synchronously to clk.
- addr (input, [31:0], —): Input signal, width [31:0].
- wdata (input, [31:0], —): AXI write data.
- wen (input, 1, —): Input signal, width 1.
- ren (input, 1, —): Input signal, width 1.
- rdata (output, [31:0], —): AXI read data.
- ready (output, 1, —): Sink-ready handshake; receiver can accept transfer when high.
