Module: i2c

Target: 32-bit RISC-V SoC — peripheral

I2C master/slave controller for sensor and EEPROM access from RV32.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- scl  inout  — inout
- sda  inout  — inout
- addr [6:0]  input  — slave addr
- data [7:0]  input  — TX byte
- rw  input  — read/write
- start  input  — input
- done  output  — output
- rdata [7:0]  output  — RX byte
- irq  output  — output

Behavior:
- Standard 100/400 kHz modes; clock stretching support.
- Memory-mapped control from APB.
