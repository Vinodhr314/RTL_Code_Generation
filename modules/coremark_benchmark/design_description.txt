Module: coremark_benchmark

Target: 32-bit RISC-V SoC — validation

Hardware performance counter snapshot block for CoreMark runs on RV32.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- start  input  — input
- stop  input  — input
- cycle_cnt [31:0]  output  — output
- instret_cnt [31:0]  output  — output

Behavior:
- Latches mcycle/minstret at CoreMark start/stop GPIO triggers.
- Read via CSR or memory map for score reporting.
