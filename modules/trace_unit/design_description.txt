Module: trace_unit

Target: 32-bit RISC-V SoC — debug

Instruction/trace export for RV32 debug (ITM-style or custom trace port).

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- pc [31:0]  input  — retired PC
- inst [31:0]  input  — retired instruction
- valid  input  — input
- trace_data [63:0]  output  — output
- trace_valid  output  — output

Behavior:
- Streams retired insn stream to off-chip buffer or SRAM ring.
- Enable via debug CSR.
