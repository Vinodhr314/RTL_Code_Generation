- clk (input, 1, —): System clock; all sequential logic sampled on rising edge unless noted.
- rst_n (input, 1, active-low): Asynchronous active-low reset; deassert synchronously to clk.
- start (input, 1, active-high): Level or pulse command to begin accelerator operation.
- op_sel (input, [3:0], —): Input signal, width [3:0].
- a (input, [31:0], —): Input port ([31:0]); see functional behavior.
- b (input, [31:0], —): Input port ([31:0]); see functional behavior.
- result (output, [31:0], —): Accelerator result word read after done.
- done (output, 1, —): Status flag: operation completed successfully.
- busy (output, 1, —): Status flag: operation in progress.
- irq (output, 1, —): Interrupt request to CPU (typically level or one-cycle pulse).
