================================================================================
5. FUNCTIONAL BEHAVIOR
================================================================================
LR loads from internal register file and sets reservation. SC stores if reservation matches addr. AMO ops apply operation to reservation register. done pulses high for one cycle when op completes. success indicates SC success.

RTL header notes:
- RV32 A-extension atomic memory operation helper
- Supports LR/SC and AMO swap/add/and/or/xor
- Verilog-2005 synthesizable
- AMO operation encoding
- Internal reservation register (one-element "register file")
- Load reservation: return current resv_data for this addr
- AMO: read-modify-write on reservation register

FSM states (from RTL localparams):
- OP_LR → OP_SC → OP_SWAP → OP_ADD → OP_AND → OP_OR → OP_XOR
