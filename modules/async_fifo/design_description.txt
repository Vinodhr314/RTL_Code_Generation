5. FUNCTIONAL BEHAVIOR
================================================================================
Gray-code pointer CDC FIFO. wfull asserted when write domain is full. rempty asserted when read domain is empty. Depth parameterized (default 16), data width 32 bits.

RTL header notes:
- Asynchronous FIFO with Gray-code pointer CDC
- Based on Cummings SNUG-2002 design; registered full/empty flags break
- the combinational loop that would exist with wire flags.
- Verilog-2005 synthesizable
- Memory array
- ----------------------------------------------------------------
- Write domain
- ----------------------------------------------------------------

FIFO operation: Push on wr_en when not full; pop on rd_en when not empty. Monitor count/empty/full in scoreboard.
