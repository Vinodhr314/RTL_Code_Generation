================================================================================
1. OVERVIEW
================================================================================
AXI4-Lite top-level 1-master/2-slave interconnect. Address bit [24] selects slave: 0=s0, 1=s1..

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Verification status: PASS

================================================================================
2. FUNCTIONAL REQUIREMENTS
================================================================================
FR-001: Module shall implement AXI4-Lite top-level 1-master/2-slave interconnect. Address bit [24] selects slave: 0=s0, 1=s1..
FR-002: Slave shall accept AXI4 write/read bursts per channel handshake rules.
FR-003: awready/wready/arready shall follow RTL timing (typically immediate or registered).
FR-004: bresp/rresp shall return OKAY (2'b00) on successful transfers.
FR-005: AXI4-Lite 1M:2S interconnect. addr[24]=0 routes to s0, addr[24]=1 routes to s1. Write sel latched on AW handshake; read sel latched on AR ha

4. TIMING, CLOCKING & RESET
================================================================================
Clock domain: single aclk (AXI).
Reset: aresetn active-low.
AXI handshakes: valid/ready per channel; respect TB task sequencing.

================================================================================
5. FUNCTIONAL BEHAVIOR
================================================================================
AXI4-Lite 1M:2S interconnect. addr[24]=0 routes to s0, addr[24]=1 routes to s1. Write sel latched on AW handshake; read sel latched on AR handshake.

RTL header notes:
- axi_top — AXI4-Lite 1-master / 2-slave interconnect (Verilog-2005)
- Address bit [24] selects slave: 0 → s0, 1 → s1
- Write-select latched on AW handshake; read-select latched on AR handshake.
- Master port (driven by CPU/DMA)
- Slave 0 port (addr[24]=0)
- Slave 1 port (addr[24]=1)
- ----------------------------------------------------------------
- Write-channel routing: latch slave select on AW handshake
