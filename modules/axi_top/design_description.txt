Module: axi_top

Target: 32-bit RISC-V SoC — bus

Top-level AXI4 interconnect wrapper for RV32 SoC masters and slaves.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- aclk  input  — input
- aresetn  input  — input
- full  AXI4  — master/slave port bundles

Behavior:
- Integrates CPU, DMA, and debug as masters; SRAM, peripherals as slaves.
- Supports INCR bursts and 32-bit data width.
