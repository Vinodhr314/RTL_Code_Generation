Module: ethernet_classifier

Target: 32-bit RISC-V SoC — network

Classifies Ethernet frames by ethertype/VLAN for steering to RV32 stacks.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- in_valid  input  — input
- in_data [31:0]  input  — input
- class [3:0]  output  — output
- class_valid  output  — output

Behavior:
- Parses DA/SA/Type fields; outputs traffic class for admission control.
- Works on streaming MAC RX path.
