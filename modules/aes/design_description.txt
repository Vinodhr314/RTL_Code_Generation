Module: aes

Target: 32-bit RISC-V SoC — crypto

AES-128/256 block encrypt/decrypt engine.

Integration context:
- Part of the RV32 platform address map and interrupt fabric
- Follows SoC coding_style.md (Verilog-2005, rst_n active-low where clocked)

Ports:
- clk  input  — input
- rst_n  input  — input
- mode  input  — enc/dec
- key [255:0]  input  — input
- block_in [127:0]  input  — input
- block_out [127:0]  output  — output
- valid  input  — input
- done  output  — output

Behavior:
- ECB block mode; CBC via software IV chaining.
- Memory-mapped key must come from secure storage.
